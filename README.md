# Edumaps_Generator
Ein leistungsstarkes, browserbasiertes Tool zum Erstellen, Bearbeiten und KI-gestützten Generieren von Edumaps für die Plattform Edumaps.de.  Dieses Tool wurde speziell für Lehrkräfte entwickelt, um Unterrichtsinhalte basierend auf dem hessischen Lehrplan (oder anderen Vorgaben) schnell in digitale Pinnwände zu verwandeln.
